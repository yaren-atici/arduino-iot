# Arduino IoT Labs – Tinkercad

A series of embedded systems experiments using Arduino Uno and Tinkercad simulator, completed as part of the Digital and Microprocessor Technology with IoT Elements course at WSB Merito Gdańsk.

## Labs Overview

### Lab 1 – Basic Digital & Analog Control
- Blinking LED
- Potentiometer reading via ADC
- RGB LED color control
- Multiple LED sequential control
- Pushbutton input handling
- Photoresistor light sensing
- TMP36 temperature sensor reading

### Lab 2 – Analog LED Control
- Potentiometer-controlled LED brightness and blinking speed
- RGB LED color selection based on voltage range (0–5V)

### Lab 3 – Sensors & Display
- HC-SR04 ultrasonic distance sensor with LED zones
- I2C LCD display showing distance in real time
- Square wave generation with oscilloscope observation
- Temperature and gas sensor with RGB warning indicators
- Binary counter with pushbuttons (increment, decrement, reset)

### Lab 4 – Keypad & Display Integration
- 4×4 keypad input detection
- Keypad + I2C LCD integration
- Keypad calculator (basic arithmetic on LCD)
- Programmable countdown timer with 7-segment display

## Tech Stack

- Arduino Uno
- Tinkercad Simulator
- C++ (Arduino framework)
- I2C communication
- PWM, ADC
- Sensors: TMP36, HC-SR04, photoresistor
- Displays: I2C LCD, 7-segment

## Platform

All experiments were simulated using [Tinkercad](https://www.tinkercad.com/) and some were also implemented on physical hardware.oT experiments — sensors, LCD, keypad, and embedded systems
